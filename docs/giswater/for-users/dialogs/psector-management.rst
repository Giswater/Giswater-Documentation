===================================
Gestor de sectores de planificación
===================================

.. only:: html

   .. contents::
      :local:

Herramienta que permite gestionar los psector. Podremos establecer como actual, hacerlo activo, restaurarlo, crear, fusionar, duplicar, mostrar o borrar un psector.
